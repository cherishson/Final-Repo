# 🎓 StudySync

A modern e-learning platform with:
- Authentication
- Courses & lessons
- Quiz system (randomized)
- Progress tracking
- AI tutor
- Payment integration (Paystack-ready)

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## 📦 Setup

### Backend
cd backend
npm install
node server.js

### Frontend
Open index.html
